Introduction: What is the essential story being told by your site and what type of structure
did you choose to implement.
The website I have built for this coursework is a magazine + news regarding the Indian Cinema
actors and actresses.
The structure of this site is divided in 5 pages. Home page has 4 separate sections, each has a
cover image with a brief introduction about a specic content. Additionally, there is a button for
each section that redirects to another page and provides detailed information on selected content
from the homepage.
State 3 things that have inspired you when creating your website.
1. Most of the inspiration came from the website of Filmafare where they have a similar
content and structure under magazine section.
2. Another website is the TimesofIndia from where most of the content is inspired.
State 3 ways in which you have considered usability in your site.
1. This website has an extra large and easily viewable content supported by images relating to
the content which in my opinion makes it easier to see for someone who has partial visuality.
State 3 things you had to learn/nd out by yourself when creating your site, and describe
how you did that.
1. First and most important learning experience was to implement multiple javascript library.
2. All the libraries implemented here has different syntax which I had to practice in order to
implement it properly.
3. Refactoring and reusing my own code from home page while maintaining similar styling
without having to create a separate style sheet for each page.
What aspect(s) of your work do you think worked well and why?
The website has more than 2 javascript libraries, all dedicated to page animations. The library
Barba worked really well with the design of the website by providing a colourful transition
animation between pages.
Another aspect that worked out wonder for this project is Scroll magic. It enabled me to
implement a modern scroll effect that is seen on many major websites such as Apple, dbrand and
many more.
What aspect(s) of your work could be improved, and how might you do things differently
another time?
There is a lot that can be improved for this website regarding the layout of the home page. The
homepage has 4 section which are style exactly the same as all other ones and could be improved
by slightly modifying the layout and implementing different structure for each section.
Another aspect is to reduce the use of multiple javascript libraries instead of which React
framework can be used to make it even more simpler in terms of code complexity while
maintaining all the features that the site already has.
What resources were used when creating your work?
1. I have implemented a transition animation between all the pages using a library called Barba.
Barba is something I had to do a lot of research and experimentation before actually
implementing it to my website.
2. Another library I used is GSAP, for animation which was actually easy to implement after
reading the documentation.
3. Finally, I have implemented a scroll effect on every page of the website for which I use a
javascript library called Scroll Magic. This was signicantly difcult to implement given
how complex it is in terms of implementation.
